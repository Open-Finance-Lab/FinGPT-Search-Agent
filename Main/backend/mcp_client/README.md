# tools

## add tools

drop .py files with `@function_tool` in `tools/`

## add mcp servers

edit `mcp_servers.json`, set `enabled: true`

## current tools

- calculator - math
- browser - playwright navigation
- images - screenshots
- pdf handler - tracks uploaded pdfs
- pageindex (mcp) - reads long pdfs

## try prompts

```
calculate 15% of $45,000
go to yahoo finance and get AAPL price
```
